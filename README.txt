# 🧩 SoftWare Engineering Project

## 🚀 Description
Ce projet est un programme Java organisé en plusieurs packages (`empaquetage` et `util`) avec une classe principale `Principal.java`.  
Il illustre la structuration d’un projet logiciel selon les bonnes pratiques de génie logiciel.

---

## ⚙️ Prérequis
Avant d’exécuter le projet, assurez-vous d’avoir :
- **Java JDK** installé (version 8 ou supérieure)
- Un terminal ou un invite de commande fonctionnel

---
